# RollScape: Enhanced Edition with Main Menu and Timer

This project is an enhanced version of the classic **RollScape** tutorial, designed to provide a more interactive and polished user experience. The enhancements include a **Main Menu**, an **In-Game Timer**, and a **Best Time Display**, making the game more engaging and user-friendly.

## Features

- **Main Menu**: 
  - Includes **Play** and **Exit** buttons for smooth navigation.
- **In-Game Timer**: 
  - Tracks the player's completion time during gameplay.
- **Best Time Display**: 
  - Automatically updates on the main menu after each win, showing the best completion time.
- **Win/Lose Detection**: 
  - The timer stops, and the game returns to the main menu upon game completion.

These features enhance the interactivity and overall experience while retaining the core gameplay of RollScape.

## Installation

The installer for the game is available in the **Releases** section of this repository. To install and play the game:

1. Navigate to the [Releases](https://github.com/vishal-sah/rollscape.git/releases) section of this repository.
2. Download the latest installer for your platform.
3. Run the installer and follow the on-screen instructions.

## How to Play

1. Launch the game and navigate through the **Main Menu**.
2. Click **Play** to start the game.
3. Control the ball to collect all the pickups as quickly as possible.
4. Your completion time will be displayed, and the best time will be saved for future runs.

## Development

This project was built using **Unity** and follows the structure of the RollScape tutorial with additional features implemented. Key scripts include:

- **BestTimeDisplay.cs**: Displays the best completion time on the main menu.
- Other scripts for timer functionality, win/lose detection, and main menu navigation.
